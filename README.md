# 📝 To-Do List (React + Vite)

A simple To-Do List application built with **React** and **Vite**.  
This project demonstrates adding, deleting, and reordering tasks with a clean UI.

---

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/to-do-list-react.git
cd to-do-list-react

2. Create project with Vite
Run the following command to set up Vite:
npm create vite@latest


- Select React
- Select JavaScript
3. Install dependencies
npm install


4. Run the development server
npm run dev


Open your browser and go to the URL shown in the terminal (usually http://localhost:5173).
